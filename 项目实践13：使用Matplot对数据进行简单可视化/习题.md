#### 1. 使用matplot绘制图片效果如下

![avatar](https://www.runoob.com/wp-content/uploads/2018/10/56F473FA-A570-4EA4-BDB5-6C82D6719FC2.png)

````
from matplotlib import pyplot as plt 
x =  [5,8,10] 
y =  [12,16,6] 
x2 =  [6,9,11] 
y2 =  [6,15,7] 
plt.bar(x, y, align =  'center') 
plt.bar(x2, y2, color =  'g', align =  'center') 
plt.title('Bar graph') 
plt.ylabel('Y axis') 
plt.xlabel('X axis') 
plt.show()
````
